## CRITICAL — Fix First (Issues #1–6)

**#1 PinManager + DuressManager — SHA-256 no salt → PBKDF2**
```java
byte[] salt = new byte[16];
new SecureRandom().nextBytes(salt);
SecretKeyFactory skf = SecretKeyFactory.getInstance("PBKDF2WithHmacSHA256");
byte[] hash = skf.generateSecret(new PBEKeySpec(pin.toCharArray(), salt, 310_000, 256)).getEncoded();
// Store salt+hash together in prefs
```

**#2 CryptoInitializer — plain SharedPrefs → EncryptedSharedPreferences**
```java
SharedPreferences prefs = EncryptedSharedPreferences.create(
    "duoshield_secure_prefs",
    MasterKeys.getOrCreate(MasterKeys.AES256_GCM_SPEC), context,
    EncryptedSharedPreferences.PrefKeyEncryptionScheme.AES256_SIV,
    EncryptedSharedPreferences.PrefValueEncryptionScheme.AES256_GCM);
```

**#3 CryptoHelper — missing UTF-8**
```java
// encrypt: plaintext.getBytes(StandardCharsets.UTF_8)
// decrypt: new String(plainText, StandardCharsets.UTF_8)
```

**#4 ChatMediaActivity.listenForMessages() — incoming messages NEVER decrypted**
```java
Boolean enc = dc.getDocument().getBoolean("isEncrypted");
if (Boolean.TRUE.equals(enc) && text != null) {
    text = EncryptionHelper.decrypt(ChatMediaActivity.this, text);
}
```

## Architecture — Strict Rules (unchanged)
1. `FLAG_SECURE` in ALL activities via `BaseActivity`
2. `FirebaseCostGuard` before every Firestore op (currently violated — fix #24)
3. One Firestore listener per screen — `onStart` attach, `onStop` detach
4. `WriteBatch` only for batch deletes (max 450 ops)
5. DiffUtil always — `setMessages()`/`setConversations()` (violated in SearchResultsAdapter — fix #50)
6. Room DB version **7** — next = v8 + MIGRATION_7_8
7. service-account.json must NEVER be in assets or git (fix #5 ASAP)
