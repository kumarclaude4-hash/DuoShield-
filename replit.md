# DuoShield

A secure, end-to-end encrypted Android messaging application with automated message self-destruction, biometric security, and duress management.

## Project Structure

- `app/` — Native Android application (Java, Gradle)
  - `src/main/java/com/duoshield/app/` — Core application logic
    - `crypto/` — ECDH key exchange + AES-GCM encryption
    - `db/` — Room local encrypted database
    - `firebase/` — Firestore, Storage, and Voice Note helpers
    - `notifications/` — FCM push notification service
    - `pairing/` — Device pairing logic
    - `security/` — Biometric auth and duress management
    - `ui/` — Activities and RecyclerView adapters
  - `src/main/res/` — XML layouts, drawables, and resources
  - `AndroidManifest.xml` — App permissions and configuration
- `functions/` — Firebase Cloud Functions (TypeScript)
  - `src/` — Cloud Function source files
  - `lib/` — Compiled JavaScript output (generated by `npm run build`)
- `firebase.json` — Firebase project configuration
- `firestore.rules` — Firestore security rules
- `firestore.indexes.json` — Firestore composite indexes

## Technologies

- **Android (Java)** — Native mobile client, minSdk 26, targetSdk 34
- **Firebase** — Firestore, Storage, Auth, Cloud Messaging (FCM)
- **Firebase Cloud Functions v2** — Scheduled cleanup and notification triggers
- **Room** — Local encrypted message persistence
- **Android Biometrics** — BiometricPrompt for app access
- **ECDH + AES-GCM** — End-to-end message encryption
- **WorkManager** — Background self-destruct scheduling
- **Gradle 8.2.0** — Android build system

## Important Notes

- This is a **native Android app** — it cannot be previewed in a browser. To run it, you need Android Studio or a connected Android device/emulator.
- Firebase Cloud Functions can be built locally with `cd functions && npm run build` and deployed with `npm run deploy` (requires Firebase CLI and project credentials).
- The app requires a `google-services.json` file (from your Firebase project) placed in `app/` to connect to Firebase services.
